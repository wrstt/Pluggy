# Support

## Getting Help

For setup help, bug reports, and usage questions, use GitHub Issues in this repository.

Maintainer profile: [SSUNOs](https://github.com/SSUNOs)

Repo issues URL: [github.com/wrstt/Pluggy/issues](https://github.com/wrstt/Pluggy/issues)

If GitHub Discussions is enabled later, general questions may be redirected there.

## Before Opening an Issue

- read `README.md`
- check existing issues for duplicates
- confirm you are using the current version/branch you intended
- remove/redact personal data, tokens, cookies, and local paths

## Include This Information

- OS and version (for example, macOS version)
- Pluggy version or commit hash
- Steps to reproduce
- What you expected
- What happened instead
- Any relevant logs/screenshots (redacted)

## Security and Privacy Issues

Do not post vulnerabilities or sensitive privacy issues publicly. See `SECURITY.md`.
